## Recommended Release Notes

`v0.1.0` is a reproducible, source-traced diagnostic release. It does not claim to solve consciousness or prove Orch-OR. It provides:

- a deterministic reproduction command
- frozen tables and summary output
- traceable literature rows
- explicit claim boundaries
